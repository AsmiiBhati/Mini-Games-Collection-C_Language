
# 🎮 C Mini Games Collection

A console-based mini games project written in C as part of my learning journey.
This repository contains a few basic games combined into a single menu-driven program.  
The focus of this project is on practicing core programming concepts.

---

## 📌 Features
The program includes:

- 🎯 Guess the Number  
  - Random number generation  
  - Limited attempts with hints  

- ❌⭕ Tic-Tac-Toe  
  - Player vs Computer  
  - Computer moves are random  

- ✊✋✌️ Rock, Paper, Scissors  
  - Simple logic against computer  
  - Randomized outcomes  

- 🧩 Hangman  
  - Word guessing with hints  
  - Limited attempts  

- 📋 Menu-driven interface to select and play games

---

## ⚙️ How to Run
Make sure you have a C compiler like GCC installed.
### Compile: 
gcc game_collection.c -o games
### Run:
**On Linux / Mac:** ./games <br>
**On Windows:** games.exe
